# SPH-Taichi

Simple 2D SPH fluid simulation implemented using [**taichi**](https://taichi.readthedocs.io/en/stable/) and python.

WCSPH and PCISPH are implemented in the project.

## Tested Environment

+ x64 windows
+ python 3
+ taichi==0.6.15
+ numpy==1.15.4
+ either GPU or CPU
